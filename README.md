# 學生學期成績管理系統 (Student Grade Management System)

## 🚀 系統核心功能

5 大核心功能與 1 項安全退出機制：

1. **新增學生資料 (Add student)**
   * 支援輸入學號、姓名及成績。
   * **重複檢查機制**：系統會自動比對現存資料，若學號重複將攔截並提示錯誤。
   * **成績區間防呆**：嚴格限制輸入成績必須在 0 ~ 100 之間。
2. **列出所有學生 (List all students)**
   * 格式化輸出目前系統內所有學生的資料。
   * 欄位自動靠左對齊，保持介面工整。
3. **依成績排序 (Sort by score)**
   * 使用高效的排序演算法（Lambda 表達式），將全班成績由**高到低**（降冪）進行排序，並自動印出排序後的列表。
4. **依學號查詢 (Search by id)**
   * 支援精確搜尋，輸入學號即可快速調出該名學生的完整詳細資料。
5. **學期成績統計 (Show statistics)**
   * 全方位編譯班級整體數據，自動計算並顯示：
     * 全班總人數
     * 班級平均分數（精確至小數點後兩位）
     * 全班最高分與最低分（**透過自訂樣版函式計算**）
     * 及格人數與不及格人數（以 60 分為及格基準）
0. **安全退出系統 (Exit)**
   * 釋放記憶體並安全結束程式。

## 🛠️ 技術特點與防呆設計

* **泛型樣版函式 (Template Functions)**：系統實作了 `getMax<T>` 與 `getMin<T>` 泛型樣版，用於動態比較班級成績並精確篩選出最高分與最低分，提升了程式碼的重用性與軟體擴充價值。
* **記憶體動態管理**：採用 `std::vector` 動態儲存學生結構體（`Student Struct`），不限制專案的學生容納人數。
* **輸入類型防呆**：當主選單要求輸入數字選項時，若使用者不小心輸入英文字母或特殊符號，系統將透過 `cin.clear()` 與 `cin.ignore()` 清除緩衝區，**不會導致程式崩潰或無窮迴圈**，並彈出錯誤提示。
* **空資料防呆**：在系統內尚未有任何學生資料時，若執行查詢、排序、統計或列出功能，系統會自動攔截並友善提示「目前無學生資料」，避免程式執行空值錯誤。
